firebird-fpc-udfs is a small freepascal library to demonstrate
how to write UDFs for Firebird in freepascal, as well as a small
collection of hopefuly useful UDFs.

Homepage:	http://code.google.com/p/firebird-fpc-udfs/

For building and installing the library read INSTALL.TXT.
For more information please visit the project's website.

You can checkout the latest version in wiki markup format via mercurial:

  hg clone https://wiki.firebird-fpc-udfs.googlecode.com/hg/ firebird-fpc-udfs-wiki

You may also want to checkout the latest sources and builds:

  hg clone https://firebird-fpc-udfs.googlecode.com/hg/ firebird-fpc-udfs
